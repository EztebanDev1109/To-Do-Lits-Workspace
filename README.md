# To-Do List App

Aplicación moderna de lista de tareas desarrollada con HTML, CSS y JavaScript Vanilla para gestionar pendientes de forma intuitiva, elegante y productiva.

## ✨ Características

- Crear tareas con:
  - Título
  - Descripción
  - Fecha límite
  - Categoría personalizada

- Gestión de categorías:
  - Crear nuevas categorías
  - Asignar colores personalizados
  - Eliminar categorías

- Filtros inteligentes:
  - Todas las tareas
  - Pendientes
  - Completadas
  - Filtrado por categoría

- Persistencia de datos con `localStorage`
- Diseño moderno tipo glassmorphism
- Animaciones suaves e interfaz responsive
- Empty states y contador dinámico de tareas

## 🛠 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage API

## 🎯 Objetivo del proyecto

Este proyecto fue desarrollado como práctica para fortalecer habilidades en:

- Manipulación avanzada del DOM
- Gestión de estado en JavaScript
- Persistencia en navegador
- Arquitectura de componentes UI sin frameworks
- Diseño UI/UX moderno
